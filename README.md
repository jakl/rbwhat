Description
-----------

Quickly list review requests that you need to respond to


Install
-------

npm install
create ~/.rbwhat.json


Run
---

./rbwhat.coffee


~/.rbwhat.json
-----------

```javascript
{
  "user": "your-reviewboard-name",
  "password": "asdfasdf",
  "url": "reviewboard.organization.com",
  "group": "optional-reviewboard-group-filter"
}
```
